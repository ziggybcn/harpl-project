Class AssemblerObj
Description:This class represents an Harpl assembler executable. When the Harpl compiler is executed, an AssemblerObj is generated as an instance of this class.
Details: This class instance can lateron be passed to the ByteCoder to generate the final executable for the Harpl Virtual Machine.
This class also contains all string contants that represent Harpl assembler instructions and instruction prefixes.
